# bdaytimer

Koack basic sample app


## Install

```sh
npm install --save bdaytimer
```

## Usage

```js
import bdaytimer from 'bdaytimer';

console.log(bdaytimer);
```

### Development

Start the run watcher

```
node watch.js
```

Watch source to build on change


```
yarn watch
```
